What is ui?
ui is a CSS/SCSS design system built for enterprise SaaS applications. It is framework-agnostic, themeable, and optimised for developer experience.
Does it require JavaScript?
Native <details> elements work without any JavaScript. For custom state classes like .is-open, a tiny JS toggle is all that is needed.
How do I apply a custom theme?
Copy themes/_company.scss, adjust the CSS variables to match your brand, and add data-theme="company" to your <html> element.
